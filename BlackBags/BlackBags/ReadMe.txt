# 전체적인 피드백

턴 표시는 되지만, 누가 누구의 땅인지, 땅 색깔과 매치가 안된다.

보드 사이즈에 따라 배경과 창 크기 싱크가 맞지 않는 증상.


assert를 사용해야 될 때와 아닐때의 구분을 잘 못하고 있다:
assert는 디버그 모드에서만 도는데, 로직의 검증 용으로 사용하고 (즉, 디버그 모드에서도 발견 가능한 로직의 검증용)

release모드에서는 다르게 동작할 수 있는 부분에선 assert를 쓰면 안된다. 특히, API나 라이브러리콜에 대해서는 직접 에러 핸들링 고고
